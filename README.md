# SurveySim

**MBES Survey Simulator** — a multibeam echosounder survey simulation tool.

---

## Download & Installation

1. Go to the **[Releases](https://github.com/josageof/surveysim-releases/releases/latest)** tab
2. Download `SurveySim Setup x.x.x.exe`
3. Run the installer and follow the on-screen instructions
4. On first launch, sign in with your access credentials

> The app checks for updates automatically — you will be notified when a new version is available.

---

## System Requirements

- Windows 10 / 11 (64-bit)
- Internet connection (required for authentication)
